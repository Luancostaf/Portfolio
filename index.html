Habilidades

Tenho mais de três anos de experiência com HTML, linguagem que aprendi de forma autodidata. Nesse tempo, desenvolvi domínio na estruturação semântica de páginas e na organização de conteúdo, sempre aplicando boas práticas para garantir clareza e acessibilidade no desenvolvimento de interfaces.

No mesmo período, aprendi e pratiquei CSS, explorando estilização avançada, responsividade e design moderno. Adquiri experiência em criar interfaces atrativas e funcionais, unindo a estrutura do HTML a um visual bem elaborado e adaptável a diferentes dispositivos.

Ao longo desses três anos, tive contato com JavaScript, explorando a linguagem em paralelo ao HTML e CSS. Embora minha experiência seja mais introdutória, busquei constantemente aprofundar meus conhecimentos para ampliar minhas habilidades no desenvolvimento front-end.

Atualmente, estudo C# no curso técnico do COTEMIG, onde tenho evoluído de forma contínua. A cada dia, adquiro novos conhecimentos sobre a linguagem, fortalecendo minha base na programação orientada a objetos e no desenvolvimento de aplicações.

Comecei a estudar MySQL neste mesmo curso técnico, e atualmente possuo conhecimento em nível intermediário. Estou em constante aprendizado, consolidando noções de modelagem de dados, criação de tabelas, uso de chaves primárias e estrangeiras, além da elaboração de consultas para manipulação e recuperação de informações.

Estou cursando Java fora do curso técnico, com o objetivo de consolidar essa linguagem como foco da minha carreira. Vejo no Java uma base robusta para atuar como desenvolvedor de software, e por isso investindo no aprofundamento de conceitos e boas práticas relacionadas ao seu ambiente.

Atualmente estudando Python no curso técnico, com base nos fundamentos da linguagem e prática em lógica de programação. Conhecimentos introdutórios voltados para compreensão de sintaxe, estruturas de controle e manipulação básica de dados.

Comecei a estudar PHP no curso técnico com o objetivo de expandir minha base no desenvolvimento back-end.
Já possuo conhecimentos iniciais sobre sua estrutura, integração com HTML e manipulação básica de dados, Buscando evoluir para a construção de aplicações web mais completas e estruturadas.
